- id: p01
  title: 逆向思维原则
  type: principle
  source_chapter: Ch2 方法论 / Ch4-演讲1(哈佛)
  source_quote: "反过来想，总是反过来想。——雅各比"
  summary: |
    解决问题的首要方法是反过来想：先研究如何失败，列出失败因素并逐一避免。芒格在哈佛演讲中用"如何确保自己过上痛苦生活"来阐述此原则，开出四味"药"：1)反复无常; 2)只从自身经验学习; 3)遭遇挫折时意志消沉; 4)忽略逆向思维。投资中同样先排除"不该做的事"，再考虑该做的事。
  tags: [逆向思维, 决策, 核心]

- id: p02
  title: 能力圈原则
  type: principle
  source_chapter: Ch2 投资评估 / Ch4-演讲2(USC)
  source_quote: "只在自己明确界定的能力圈内行事"
  summary: |
    明确知道自己真正理解的领域边界，只在圈内行动。投资时将项目分为三类："可以投资""不能投资""太难理解"。不清楚的领域直接跳过，宁可错过也不做超出能力圈的事。查理将制药业和高科技归入"太难理解"，大张旗鼓的交易归入"不能投资"。
  tags: [能力圈, 投资, 边界]

- id: p03
  title: 安全边际原则
  type: principle
  source_chapter: Ch2 投资检查清单-风险
  source_quote: "测算合适的安全边际"
  summary: |
    所有投资评估应从测量风险开始。买入价格必须远低于内在价值，为判断错误和不可预见风险留出保护空间。同时坚持为预定风险要求合适的补偿，永远记住通货膨胀和利率的风险。避免犯下大错，避免资本金持续亏损。
  tags: [安全边际, 投资, 风险管理]

- id: p07
  title: 严格分析与检查清单原则
  type: principle
  source_chapter: Ch2 投资检查清单-严格分析
  source_quote: "使用科学方法和有效的检查清单能够最大限度地减少错误和疏忽"
  summary: |
    区分价值和价格、过程和行动、财富和规模。记住浅显的好过掌握深奥的。成为商业分析家而非市场分析家。考虑总体的风险和效益，永远关注潜在的二阶效应和更高层次的影响。要朝前想、往后想——反过来想，总是反过来想。将所有思维模型编排为检查清单，系统化地避免错误。
  tags: [分析, 检查清单, 二阶效应]

  tags: [耐心, 复利, 纪律]

- id: p09
  title: 集中投资与好球区原则
  type: principle
  source_chapter: Ch2 投资评估 / Ch3 即席谈话
  source_quote: "我们跨不过七英尺高的栏。我们寻找的是那些一英尺高的、对面有丰厚回报的栏"
  summary: |
    投资成功来自少数几次重大决策。用泰德·威廉姆斯的击球哲学：把击打区划分为77个格子，只有球落在最佳格子时才挥棒。大多数时候什么也不做，只观察。当速度很慢、线路又直的好球落在最爱的格子中间时，全力出击。伯克希尔最成功的10个投资项目就足以让两辈子花不完。
  tags: [集中投资, 机会成本, 等待]

- id: p10
  title: 价值优于价格原则
  type: principle
  source_chapter: Ch2 投资评估
  source_quote: "股价公道的伟大企业比股价超低的普通企业好"
  summary: |
    这是芒格对格拉汉姆价值投资的关键修正。重要的不是以超低价买入平庸企业，而是以合理的价格买入伟大企业。整个评估过程的最终目标是对比价值(你得到的)和价格(你付出的)。财务报告只是起点而非终点，需要重新调整所有数字以反映真实价值。
  tags: [价值投资, 价格, 护城河]

- id: p11
  title: 护城河原则
  type: principle
  source_chapter: Ch2 投资评估 / 伯克希尔年会
  source_quote: "把护城河、保持它的宽度和保持它不被跨越作为伟大企业的首要标准"
  summary: |
    企业的竞争优势就是它的"护城河"。优秀的公司拥有很深的护城河，这些护城河不断加宽，为企业提供长久保护。要从方方面面——产品、市场、商标、雇员、分销渠道、社会潮流——评估竞争优势及其持久性。努力识别且只购买那些有很大机会击败"竞争性毁灭"力量的企业。
  tags: [护城河, 竞争优势, 企业分析]

- id: p15
  title: 双轨分析原则
  type: principle
  source_chapter: Ch2 方法论
  source_quote: "首先，理性地看，哪些因素真正控制了涉及的利益；其次，有哪些潜意识因素会使大脑自动形成结论"
  summary: |
    决策时使用双轨分析：第一轨是理性分析法，认准真正的利益和机会；第二轨是评估那些造成潜意识结论的心理因素(大多数是错误的)。理性分析处理可量化的利益关系，心理分析防范25种误判心理倾向的影响。两轨并行才能做出高质量决策。
  tags: [双轨分析, 理性, 潜意识]

- id: p17
  title: "五 W 原则"(布劳恩沟通法则)
  type: principle
  source_chapter: Ch4-演讲2(USC) / 演讲11
  source_quote: "你必须讲清楚何人将在何时何地因何故做何事"
  summary: |
    卡尔·布劳恩的管理规矩：任何指令必须讲清楚何人(Who)在何时(When)何地(Where)因何故(Why)做何事(What)。如果给属下写纸条吩咐他做事情却没有交待原因，布劳恩可能会解雇你。人们只有一丝不苟地把某个想法的原因都摆出来，这个想法才最容易被接受。
  tags: [沟通, 管理, 执行]

- id: p28
  title: 坐等投资法(不卖原则)
  type: principle
  source_chapter: Ch2 投资评估
  source_quote: "我们偏向于把大量的钱投放在我们不用再另外作决策的地方"

    防止一种习惯的养成要比改变它容易得多。在生活中维持许多好习惯、避免或戒除许多坏习惯的生活才是明智的。大脑的抗改变倾向使得改掉坏习惯极为困难。应在年轻时就开始培养好习惯，因为三岁看老不无道理。
  tags: [习惯, 预防, 自律]

- id: p58
  title: 间接教育法原则
  type: principle
  source_chapter: Ch4-演讲3(斯坦福)
  source_quote: "如果这个道理是我自己摸索出来的，我会记得更牢"
  summary: |
    芒格父亲用间接方法教他做人的道理——不直接灌输而是让他自己思考领悟。如果你动脑筋才懂得某个道理，就会更好地记住它(承诺和保持一贯性倾向)。优秀的文学作品需要读者略加思索才能理解，那样影响会更深。教育孩子和下属时，间接引导比直接说教更有效。
  tags: [教育方法, 间接引导, 记忆]

- id: p59
  title: 顺应优势原则(选择你的游戏)
  type: principle
  source_chapter: Ch4-演讲3 Q&A
  source_quote: "你们每个人都必须搞清楚你们有哪方面的才能。你们必须发挥自己的优势"
  summary: |
    必须发挥自己的优势，不要在较不擅长的领域寻求成功。芒格和巴菲特不投资高科技企业，因为他们宁愿在有优势的领域游戏，也不去竞争激烈且毫无优势甚至处于劣势的领域游戏。如果在较不擅长的领域取得成功，那要么是中了彩票，要么就是极其走运。
  tags: [优势, 定位, 竞争]

- id: p60
  title: 面对错误原则(学会放弃)
  type: principle
  source_chapter: Ch4-演讲3 Q&A
  source_quote: "生活有时候就像扑克游戏，有时候即使拿到一把非常喜欢的牌，也必须学会放弃"
  summary: |
    如何对付错误和改变赢面的新情况是必须掌握的知识。人们破产的通病是无法正确处理心理否认——投入越多，一贯性原理就越促使继续投入。剥夺性超级反应也会出现。必须学会在情况不利时停下来反思，从头再来，而不是执迷不悟导致破产。
  tags: [放弃, 错误处理, 止损]
