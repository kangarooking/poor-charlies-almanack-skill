# 反衍生品原则 — 淘汰原因

- id: p30
  merged_from: [p30, ce32, c13]
  type: principle

- V1: 通过 — 第三讲(即席谈话)、ce32、c13(预测2008危机)中均有讨论。
- V2: 不通过 — "避免衍生品"是一个具体的投资建议,不是通用的分析方法论。用在新场景(比如评估一家使用衍生品对冲风险的公司)中,只能得出"小心"这种泛泛结论。
- V3: 不通过 — "衍生品是大规模杀伤性金融武器"虽表述生动,但"复杂金融产品有风险"是基本常识。2008年金融危机已使此观点成为主流。
- 淘汰原因: V2+V3双重不通过。是具体领域的观点,不是通用方法论。
