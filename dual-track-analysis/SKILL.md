# 双轨分析 (Dual-Track Analysis)

## R — 原文 (Reading)
> "从个人的角度来讲,我已经养成了使用一种双轨分析的习惯。首先,理性地看,哪些因素真正控制了涉及的利益?其次,当大脑处于潜意识状态时,会自动形成哪些潜意识因素——这些潜意识因素总的来讲有用但又常常失灵?"
>
> — 查理·芒格, 第二章(芒格的生活、学习和决策方法)

---

## I — 方法论骨架 (Interpretation)
任何决策都应同时在两条轨道上运行。第一轨是理性分析——识别哪些因素真正控制了
所涉及的利益,用概率、数学和逻辑进行推算。第二轨是心理学分析——审视自己和
他人的潜意识中,有哪些心理倾向可能扭曲判断。核心洞察:即使在做"理性"分析时,
认知偏差也在暗中起作用,理性轨道本身需要被审查。大多数人完全忽视第二轨,
或将其简单化为"跟着感觉走"。但第二轨是系统化的(25种倾向清单),不是模糊
的"直觉"。芒格自称此方法使他"超越那些比他聪明得多的人"。

## A1 — 书中的应用 (Past Application)
### 案例 1: 芒格的投资决策实践
- **问题**: 如何在投资中持续做出高质量判断
- **方法论的使用**: 芒格自称在每次投资决策中始终使用双轨分析——第一轨分析企业价值、竞争格局、管理层质量;第二轨逐项检查自己是否受到25种心理倾向的影响
- **结论**: 理性分析处理可量化的利益关系,心理分析防范潜意识中的判断扭曲
- **结果**: 芒格称此方法使他"超越比他聪明得多的人,还能赚到很多钱"

### 案例 2: 可口可乐思维实验中的双轨运用
- **问题**: 如何用200万美元本金建立价值2万亿美元的饮料公司
- **方法论的使用**: 同时用理性分析(经济学/规模优势)和潜意识分析(条件反射如何影响消费者行为、社会认同如何驱动品牌忠诚)
- **结论**: 成功的商业策略必须同时满足理性经济逻辑和人类心理倾向
- **结果**: 推演出了一个与可口可乐实际发展高度吻合的商业蓝图

### 案例 3: 芒格推翻两组工程师方案
- **问题**: 面对两组对立工程师提出的技术方案,如何判断谁对
- **方法论的使用**: 第一轨:用工程学知识理性评估两种方案的技术优劣;第二轨:识别两组工程师各自存在的激励偏见(每组方案都对自身有利)
- **结论**: 方案质量不仅取决于技术分析,还取决于提出者的激励结构
- **结果**: 找到第三位无利益冲突的工程师,获得了更安全、更便宜的方案

## A2 — 触发场景 (Future Trigger) ★
### 用户会在什么情境下需要这个 skill?
1. 做出了一个逻辑严密的分析,但直觉感到不安或"太好了以至于不真实"
2. 需要评估他人(顾问、专家、上级)给出的建议是否受到利益扭曲
3. 在重大决策前需要系统性地自检是否存在认知偏差
4. 面对群体决策时,需要识别团队中可能存在的集体心理偏差

### 语言信号 (用户的话里出现这些就应激活)
- "我的分析逻辑上没问题,但总觉得哪里不对"
- "这个建议看起来很合理,但我总觉得有偏见"
- "他/她的推荐是不是有利益关系"
- "我好像太乐观了/太悲观了,但说不清为什么"
- "帮我检查一下我的判断有没有偏差"
- "我的直觉和分析结论矛盾"

### 与相邻 skill 的区分
- 与 `inversion-thinking` 的区别: 逆向思维是**方向反转**——把"怎么成功"翻转为"怎么失败";双轨分析是**轨道并行**——同时检查理性逻辑和潜意识偏差。逆向思维关注方案本身的漏洞,双轨分析关注**判断者自身的偏差**。
- 与 `latticework-mental-models` 的区别: 多元思维模型是自由组合任意多学科模型;双轨分析固定使用**理性+心理学**两条轨道。双轨分析可以理解为多元思维模型在决策自检场景下的一个特定应用模式。

## E — 可执行步骤 (Execution)
当 skill 被激活后, agent 应按以下步骤执行:

1. **第一轨: 理性分析**
   - 完成标准: 列出真正控制所涉及利益的关键因素,用概率和逻辑评估各选项的期望结果。判停点:如果理性分析本身的数据不足,先补充数据再继续

2. **第二轨: 心理倾向自检**
   - 完成标准: 逐一检查以下高发心理倾向是否在影响判断(不必全部25种,优先检查与当前场景最相关的):
     - 激励机制偏见: 分析者/建议者是否因结论而获利?
     - 社会认同: 是否因为"大家都这么想"而接受?
     - 避免不一致性: 是否因为前期已投入而拒绝改变?
     - 过度乐观: 是否只看了最好情景?
     - 喜欢/热爱或讨厌/憎恨: 是否因情感而扭曲了评估?
     每项检查必须给出"是/否/不确定"判断和简要理由

3. **两轨交叉验证**
   - 完成标准: 将理性分析结论与心理倾向检查结果对比。如果两轨指向一致(理性结论可靠+无重大偏差),信心增强;如果冲突(理性结论看起来好但发现重大偏差),标记为高风险判断。判停点:如果发现2个及以上心理偏差在影响判断,必须暂停并重新审视理性分析的前提假设

4. **输出修正后的决策建议**
   - 完成标准: 明确说明理性分析结论是否被心理倾向修正,修正后的最终判断是什么。如果心理倾向检查发现了重大偏差,必须给出具体的偏差缓解措施

## B — 边界 (Boundary) ★
### 不要在以下情况使用此 skill
- 纯技术/数学问题中不存在人类判断偏差(如代码语法错误、数学计算)
- 紧急决策中来不及做系统性双轨检查(此时应依赖预制的检查清单快速判断)
- 用户明确只需要理性分析维度,且主动声明不考虑心理偏差因素

### 作者在书中警告的失败模式
- 喜欢/热爱导致的认知扭曲: "爱"上某个方案/投资后不愿放弃,自动忽略其缺点(ce04)
- 讨厌/憎恨导致的认知扭曲: 因讨厌某人/某方案而自动忽略其优点(ce05)
- 受简单联想影响的误判: 过去成功经验导致错误联想,认为这次也必然成功(ce08)
- 自视过高倾向: 过度自信于自己的理性分析能力,高估自己对偏差的免疫力(ce10)

### 作者的盲点 / 时代局限
- 芒格和巴菲特的成功极度依赖美国20世纪后半叶的特殊历史机遇,双轨分析的效果可能被夸大
- 作为超级富豪,对普通人面临的约束(信息不对称、资本规模、时间压力)可能缺乏感知
- 25种心理倾向体系是个人整理,未经过严格的学术实证验证

### 容易混淆的邻近方法论
- "相信直觉"是模糊的情感判断;双轨分析的**第二轨是系统化的**(25种倾向逐项检查),不是"跟着感觉走"
- "魔鬼代言人"(故意唱反调)是外部对抗;双轨分析是**内在自检**,不需要外部角色扮演
